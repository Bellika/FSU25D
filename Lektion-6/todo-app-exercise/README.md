# Todo App - Övningsprojekt

Detta är övningsprojektet för Lektion 6 där du ska bygga en komplett Todo-app.

## Kom igång

```bash
npm install
npm run dev
```

## Projektstruktur

Projektet har en grundläggande struktur:
- `src/App.jsx` - Huvudkomponent som renderar TodoApp
- `src/components/TodoApp.jsx` - Din Todo-app (nästan tom - här börjar du!)

## Din uppgift

Följ instruktionerna i `src/components/TodoApp.jsx` för att bygga upp din Todo-app.

Du kommer att skapa:
1. **Form-komponent** - För att lägga till nya todos
2. **TodoList-komponent** - För att visa listan av todos
3. **Todo-komponent** - För att visa en enskild todo
4. **Header-komponent** - (Del 4)
5. **Footer-komponent** - (Del 4)

## Funktionalitet att implementera

- ✅ **Create** - Lägg till nya todos
- ✅ **Read** - Visa todos i en lista
- ✅ **Update** - Ändra befintliga todos
- ✅ **Delete** - Ta bort todos (Del 1)
- ✅ **Complete** - Markera todos som färdiga (Del 2)
- ✅ **Filter** - Separata listor för completed/incomplete (Del 3)
- ✅ **Layout** - Header och Footer (Del 4)
- ✅ **Styling** - CSS Modules (Del 5)
